# Tooftd
Tooftd is a innovative new furniture system designed around social interaction and comfort. Instead of creating furniture centered around technology, like a TV or tables for laptops, Tooft helps to bring people together.

This Github repo serves as a space where the code for the currently work-in-progress Tooft website is being stored. This website will include a landing page, team bios, and a mock check-out flow.

<img width="1530" height="359" alt="image" src="https://github.com/user-attachments/assets/56cd9829-f07c-4456-a327-5ca2fc40f053" />

## Technology
This website is being created with the following technologies:
- HTML, CSS, JS
- React (JavaScript Framework)
- Vite
- Vercel (Hosting)

We are also using Figma for design and Claude Code to aid in development.
